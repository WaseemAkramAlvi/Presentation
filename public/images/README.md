# Images Folder

Please add the following image files to this folder:

1. **punjablogo.png** - Punjab Government official logo
2. **tevtalogo.png** - TEVTA official logo  
3. **hero-bg.jpg** - Background image for the title slide

These images will be automatically loaded by the application.

## Supported Formats
- PNG (recommended for logos)
- JPG/JPEG (recommended for backgrounds)
- SVG (alternative for logos)
